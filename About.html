About

Development of Website

The intended purpose for this website was for the high school webmaster event. This website provides information on Augmented Reality, MOT High, and our TSA Chapter. The website domain was registered with domains.google.com. Everything on this website was coded from scratch using the languages of HTML and CSS on CodeAnywhere. Pre-made website templates were NOT used. The code is stored in github.io. All information was collected by experience and research by the team.
